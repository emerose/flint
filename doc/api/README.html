## Flint — a firewall rule analyzer

Supported firewalls:

  • Cisco Pix/ASA
  • IOS almost...
  • IPTables soon
  • Pf soon

## Installing

Needs the following gems:

From our private git repos

  • matasano_utils — ssh://www.runplaybook.com/var/git/matasano_utils.git
  • ralex — ssh://www.runplaybook.com/var/git/ralex.git

git clone them, rake gem; gem install; ./pkg/*.gem

Public:

  • sinatra
  • haml
  • ohm
  • rdiscount

You will also need to download, compile and run redis.

## Developers

You will need the `racc` gem, and Ragel installed. Ragel is available at: www.complang.org/ragel/

## Project Layout

  • `lib`

    Ruby code for the Flint module, as well as the support modules for specific firewall types.

  • `app`

    The sinatra app which provides an interface to Flint.

  • `checks`

    Contains all of firewall rule checks that Flint uses. Each firewall has it’s own subdir for example: 

        checks/<firewall>/*.ftg

    Rule checks are defined in Flint Test Group files, each one defines a specific test group, and it’s associated checks. The `checks/test` subdir contains a set of dummy tests which exercise the UI and the Flint test result classes.

  • spec

    RSpec tests for Flint.
